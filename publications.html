Code and Data

  • MATLAB Code for Computing Second-Order Voltage Sensitivities
    This MATLAB package provides functions to compute the second-order sensitivity matrix of bus voltages with respect to active and reactive power injections. The code implements the analytical framework presented in our paper:
    "Adaptive Power Flow Approximations With Second-Order Sensitivity Insights", IEEE Transactions on Power Systems, vol. 40, no. 3, pp. 2648–2660, May 2025.
